🎓 Research & Academic Projects

Enhanced Cardless Transaction Using Biometric ATMs

Designed and developed a biometric-based ATM authentication system that eliminates the need for traditional ATM cards and PINs by utilizing fingerprint and facial recognition technologies.

  • Implemented a secure authentication mechanism using fingerprint and face recognition for user verification.
  • Utilized Raspberry Pi as the controller to process biometric inputs and interact with the database.
  • Applied OpenCV libraries and Haar Cascade algorithms for facial recognition, verification, and identification.
  • Focused on improving transaction security and reducing risks associated with lost cards and compromised PINs.
  • Successfully published both a survey paper and a journal paper based on the project.

Merged Deep BI-GRU and CNN Model for Supervised Drug Based Sentiment Analysis with Question-Answering Mechanism

Developed a hybrid deep learning framework combining Bidirectional Gated Recurrent Unit (Bi-GRU), Convolutional Neural Networks (CNN), and Spark NLP Question-Answering techniques to analyze sentiment in drug reviews and provide intelligent medicine-related responses.

  • Designed an end-to-end NLP pipeline for drug review preprocessing, sentiment classification, and question answering.
  • Integrated Bi-GRU for contextual language understanding and CNN for feature extraction to improve sentiment prediction performance.
  • Implemented Spark NLP Question-Answering mechanisms to retrieve relevant responses from review datasets based on user queries.
  • Achieved 93.13% validation accuracy, outperforming multiple traditional machine learning approaches including Random Forest, Naïve Bayes, SVM, Decision Tree, and KNN.
  • Focused on helping users understand medication effectiveness, experiences, and potential side effects through review analysis.
